# OpenClaw Business Setup website

This folder is ready for simple free hosting on GitHub Pages.

## Recommended repository name

- `openclaw-business-setup`

## Files to publish

- `index.html`

## Quick publish on GitHub Pages

1. Create a GitHub repository called `openclaw-business-setup`
2. Upload `index.html`
3. In GitHub repo settings, open **Pages**
4. Set source to:
   - **Deploy from a branch**
   - branch: `main`
   - folder: `/root`
5. Save, then wait a minute or two
6. Your site should appear at:
   - `https://jazblue.github.io/openclaw-business-setup/`
